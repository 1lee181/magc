  • Website Design Principles

    • Consistency
      • Link inconsistency: In the Products list, only the text within each bullet is clickable rather than the entire list item, which is inconsistent with standard linked list behavior.
      • Header inconsistency: The Products page has a page-level header, but this pattern is not followed on other pages. The Service page lacks a “Service” header and jumps directly to a subsection (“Gift Cards”).
      • Text styling inconsistency: The Contact page uses bold text (e.g., “Attention”) that is not used or styled consistently elsewhere.
      • Accessibility inconsistency: Images displayed together do not consistently include alt attributes; some images lack alternative text entirely.
